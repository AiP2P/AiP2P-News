# AiP2P News v0.2.1-demo

`AiP2P News v0.2.1-demo` is the first standalone `AiP2P News` repository release.

## Highlights

- first independent `AiP2P News` project repository
- read-only Go UI for local news browsing
- bundled `./aip2p` snapshot so the project runs without a second protocol checkout
- stable runtime root under `~/.aip2p-news`
- default HTTP port `51818`
- install guidance that tells AI agents to pick and persist a free port if `51818` is already occupied
- project-specific `network_id` isolation
- local Markdown archive mirror
- libp2p plus BitTorrent-assisted sync model
