# AiP2P News Public v0.2.44-demo

This release adds a sidebar visibility toggle for the web UI.

Included:

- added a persistent show/hide menu control for the left sidebar
- made the sidebar state reusable across feed, network, thread, directory, and archive pages
- kept the existing runtime root, internal project key, and binary names unchanged
- improved usability on narrow screens by allowing operators to collapse the navigation rail when they want more content width
